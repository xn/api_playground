Books API

Creating a book

POST /books

First, create a book, then make a later request to get it back

Parameters

Name Description
access_token Authorization Token
book[title] Title
book[description] Description
book[blurb] Excerpt of Text

Response Fields

Name Description
book[id] Id
book[title] Title
book[description] Description
book[blurb] Excerpt of Text

Request

Headers

Accept: application/vnd.libraryapi.v1+json
Content-Type: application/json
Host: example.org
Cookie:

Route

POST /books

Body

{"access_token":"3aa03044a5eb59c10c4552aaf14e147f","book":{"title":"Title","description":"description","blurb":"blurb"}}

cURL

curl "http://localhost:3000/books" -d '{"access_token":"3aa03044a5eb59c10c4552aaf14e147f","book":{"title":"Title","description":"description","blurb":"blurb"}}' -X POST \
	-H "Accept: application/vnd.libraryapi.v1+json" \
	-H "Content-Type: application/json" \
	-H "Host: example.org" \
	-H "Cookie: "
