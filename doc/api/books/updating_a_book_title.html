Books API

Updating a book title

PUT /books/:id

Parameters

Name Description
id Id
access_token Authorization Token
book[title] Title
book[description] Description
book[blurb] Excerpt of Text

Response Fields

Name Description
book[id] Id
book[title] Title
book[description] Description
book[blurb] Excerpt of Text

Headers

Accept: application/vnd.libraryapi.v1+json
Content-Type: application/json
Host: example.org
Cookie:
